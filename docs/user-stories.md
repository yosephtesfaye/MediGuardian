# User Stories

## Patient

As a patient,

I want to register my medications

so that I never forget them.

---

As a patient,

I want reminder notifications

so I take medicine on time.

---

As a patient,

I want the AI to remember my schedule

so I don't repeat information.

---

## Caregiver

As a caregiver,

I want alerts when medication is repeatedly missed

so I can follow up.

---

## Administrator

As an administrator,

I want analytics

so I can monitor adherence.

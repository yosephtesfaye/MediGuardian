# Functional Requirements

The system shall:

FR-1 Register medications.

FR-2 Update medications.

FR-3 Delete medications.

FR-4 Generate reminder schedules.

FR-5 Send reminders.

FR-6 Track medication adherence.

FR-7 Store user preferences.

FR-8 Remember caregivers.

FR-9 Notify caregivers.

FR-10 Generate adherence reports.

FR-11 Reject unsafe medical advice.


# Non-Functional Requirements

Performance

- Response < 5 seconds

Security

- API keys stored in environment variables

Availability

- Cloud Run deployment

Scalability

- Support additional agents

Maintainability

- Modular architecture

Reliability

- Persistent storage

Usability

- Easy conversation

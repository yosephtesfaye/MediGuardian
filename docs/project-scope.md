# Project Scope

## Overview

MediGuardian AI is a Concierge AI Agent designed to help users organize, remember, and monitor their medication schedules.

The application focuses on:

- Medication organization
- Daily reminders
- Adherence tracking
- Caregiver communication
- Intelligent conversation
- Safe AI interactions

---

## In Scope

✔ Medication scheduling

✔ Natural language understanding

✔ Reminder generation

✔ Caregiver notifications

✔ Memory

✔ Analytics

✔ Multi-Agent workflow

✔ REST API

✔ Cloud deployment

---

## Out of Scope

❌ Disease diagnosis

❌ Prescription generation

❌ Dosage modification

❌ Emergency medical decisions

❌ Drug recommendation
